Análisis de las Tareas


Ingresar a la aplicación y registrarse

  1. Caja de texto para usuario.
  2. Etiqueta usuario.
  3. Caja de texto password.
  4. Botón de registro.
Acciones
  1. Ingresar valores alfanuméricos en la caja de texto usuario, mediante el teclado del dispositivo.
  2. Ingresar valores alfanuméricos en la caja de texto password, mediante el teclado del dispositivo.
  3. Seleccionar el botón registro mediante el dispositivo.

Tarea 2: Seleccionar el nivel de dificultad del videojuego.
  1. Botón 1: Dificultad fácil.
  2. Botón 2: Dificultad media.
  3. Botón 3: Dificultad difícil.
  4. Botón 4: Salir.

Acciones:
  1. Seleccionar botón Dificultad fácil mediante el dispositivo.
  2. Seleccionar botón Dificultad media mediante el dispositivo.
  3. Seleccionar botón Dificultad difícil mediante el dispositivo.
  4. Seleccionar botón Salir mediante el dispositivo.

Tarea 3: Configurar resolución.
  1. Botón 1: Alta
  2. Botón 2: Media
  3. Botón 3: Baja
  4. Botón 4: Cerrar

Acciones:
  1. Seleccionar botón Alta mediante el dispositivo.
  2. Seleccionar botón Media mediante el dispositivo.
  3. Seleccionar botón Baja mediante el dispositivo.
  4. Seleccionar botón Cerrar mediante el dispositivo.

Tarea 5: Jugar
  1. Botón flecha arriba.
  2. Botón flecha abajo.
  3. Botón flecha izquierda.
  4. Botón flecha derecha.
  5. Botón rojo.
  6. Botón azul.

Acciones:
  1. Seleccionar botón flecha arriba mediante el dispositivo para mirar hacia arriba.
  2. Seleccionar botón flecha abajo mediante el dispositivo para agacharse.
  3. Seleccionar botón flecha izquierda mediante el dispositivo caminar hacia atrás.
  4. Seleccionar botón flecha derecha mediante el dispositivo caminar hacia adelante.
  5. Seleccionar botón rojo mediante el dispositivo para saltar.
  6. Seleccionar botón azul mediante el dispositivo para hacer una acción"
